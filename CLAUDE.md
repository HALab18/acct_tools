# acct_tools — CLAUDE.md

## プロジェクト概要

日本の税制・給与計算に特化したスタンドアロン Web ツール集。
すべてブラウザで直接開ける単一 HTML ファイル（ビルド不要）。

## ファイル構成

```
acct_tools/
└── src/
    ├── Furusto_Rimit_tax.html      ふるさと納税 寄付上限額 算定ツール v2
    ├── interest_calculator_v4.html  法人版 利子所得税 計算シート v4
    └── tedori_gyaku_v3.html         手取り逆算ツール｜給与計算 v3
```

## 各ツールの役割

| ファイル | 目的 | テーマ |
|---|---|---|
| Furusto_Rimit_tax.html | 給与収入または住民税額からふるさと納税の上限額を算出 | ダーク（ゴールド/ブルー） |
| interest_calculator_v4.html | 差引入金額 ↔ 税引前利息の相互変換（税率 15.315%） | ライト（ブルー/ホワイト） |
| tedori_gyaku_v3.html | 総支給額 → 手取り、または希望手取り → 必要総支給額の逆算 | ライト（クリーム/グリーン） |

## 技術スタック

- **Vanilla JavaScript** — フレームワーク・ライブラリなし
- **HTML5 / CSS3** — CSS カスタムプロパティ（変数）で全テーマ管理
- **Google Fonts** — Noto Sans JP, Noto Serif JP, IBM Plex Mono, Shippori Mincho, DM Mono
- **外部依存なし** — npm / ビルドツール不要

## コーディング規約

### 命名
- 計算変数は日英混在（例: `juminzei`, `kyuyo`, `gensen`）
- DOM イベントは `on` プレフィックス（`onSalaryChange`, `onCommaInput`）
- フォーマット系ユーティリティは短縮名（`fmt()`, `addCommas()`, `numVal()`）

### CSS
- テーマカラーはファイル先頭の `:root` に集約
- ダークテーマ: `--bg #0c1620`, `--gold #f0dc9a` 系
- ライトテーマ: `--accent #2563eb` または `--accent #4a7c59` 系
- スペーシング: 16 / 20 / 28 / 36px 基準
- ブレークポイント: 480px, 560px, 620px

### HTML
- 1ファイル完結（CSS・JS はすべてインライン）
- `id` / `class` セレクタ中心（`data-*` 属性は最小限）
- リアルタイム計算は `oninput` で発火（ボタン押下不要）

## 日本税制の主要ロジック

- **ふるさと納税上限** = 住民税所得割額 × 20% ÷ (0.9 − 所得税率 × 1.021) + 2,000円
- **利子所得税率** = 15.315%（所得税 15% ＋ 復興特別所得税 0.315%）
- **標準報酬月額** は等級テーブルから参照（健康保険・厚生年金）
- **源泉徴収** は扶養親族数を考慮した税額表で算出
- 税率・控除額は年度改定があるため、変更時はファイル内テーブルを直接更新する

## 新規ツール追加時のガイドライン

1. `src/` 直下に `<機能名>_v1.html` として配置
2. 既存ファイルの CSS 変数・関数命名スタイルに合わせる
3. `@media print` スタイルを必ず含める（印刷・PDF 出力対応）
4. モバイル対応（480px ブレークポイント）を必須とする
5. 外部 CDN への依存は追加しない

## バージョン管理

- ファイル名末尾の `_v{N}` でバージョン管理（例: `tedori_gyaku_v3.html`）
- 破壊的変更時はバージョン番号をインクリメントして新ファイルを作成
- 旧バージョンは履歴として `src/` に残す（削除しない）
