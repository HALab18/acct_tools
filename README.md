# acct_tools

日本の税制・給与計算に特化したスタンドアロン Web ツール集。  
すべてブラウザで直接開ける単一 HTML ファイルです。ビルド不要・依存ライブラリなし。

---

## ツール一覧

| ファイル | ツール名 | 概要 |
|---|---|---|
| [`src/Furusto_Rimit_tax.html`](src/Furusto_Rimit_tax.html) | ふるさと納税 寄付上限額 算定ツール v2 | 給与収入または住民税額からふるさと納税の上限額を算出 |
| [`src/interest_calculator_v4.html`](src/interest_calculator_v4.html) | 法人版 利子所得税 計算シート v4 | 差引入金額 ↔ 税引前利息の相互変換（税率 15.315%） |
| [`src/tedori_gyaku_v3.html`](src/tedori_gyaku_v3.html) | 手取り逆算ツール \| 給与計算 v3 | 総支給額 → 手取り、または希望手取り → 必要総支給額の逆算 |

## 使い方

1. リポジトリをクローンまたは ZIP ダウンロード
2. `src/` 内の HTML ファイルをブラウザで直接開く

```bash
git clone https://github.com/fetofeto2-lab/acct_tools.git
cd acct_tools/src
# ブラウザで開く（例: macOS）
open Furusto_Rimit_tax.html
```

サーバー不要。インターネット接続は Google Fonts の読み込みのみに使用します。

## 各ツールの詳細

### ふるさと納税 寄付上限額 算定ツール v2

給与収入または住民税所得割額を入力するだけで、ふるさと納税の自己負担 2,000 円となる上限額を自動計算します。

**計算ロジック**
```
上限額 = 住民税所得割額 × 20% ÷ (0.9 − 所得税率 × 1.021) + 2,000円
```

- テーマ: ダーク（ゴールド / ネイビー）
- 2 つの入力モード: 給与収入入力 / 住民税額直接入力

---

### 法人版 利子所得税 計算シート v4

定期預金などの利子に対する源泉徴収税額（15.315%）を双方向で計算します。

**計算ロジック**
```
利子所得税率 = 15.315%（所得税 15% + 復興特別所得税 0.315%）
差引入金額 → 税引前利息: 入金額 ÷ (1 − 0.15315)
税引前利息 → 差引入金額: 利息 × (1 − 0.15315)
```

- テーマ: ライト（ブルー / ホワイト）
- 複数行の一括入力に対応

---

### 手取り逆算ツール | 給与計算 v3

社会保険料・源泉所得税・住民税を考慮した正確な手取り計算と逆算を行います。

**対応計算**
- 総支給額 → 手取り額
- 希望手取り額 → 必要総支給額（逆算）

**考慮項目**
- 健康保険料（標準報酬月額テーブル参照）
- 厚生年金保険料
- 雇用保険料
- 源泉所得税（扶養親族数考慮）
- 住民税

- テーマ: ライト（クリーム / グリーン）

## 注意事項

- 税率・保険料率・控除額は年度改定により変更される場合があります。最新の法令に基づいてご確認ください。
- 本ツールの計算結果は参考値です。正確な税務処理については税理士等の専門家にご相談ください。
